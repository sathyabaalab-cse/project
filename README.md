# 7-Day AI/ML Micro-Internship

## Project Description
This repository contains the Google Colab notebooks and source code developed during my **7-Day AI/ML Micro-Internship**. This intensive short-term program focused on building a foundational understanding of Artificial Intelligence and Machine Learning concepts, data preprocessing techniques, and practical implementation of basic ML models.

## Key Learnings & Implementations
- **Data Exploration:** Learned how to load, clean, and manipulate datasets efficiently using standard data science libraries.
- **Data Visualization:** Generated insightful charts and graphs to identify trends, correlations, and anomalies in the data.
- **Model Development:** Built, trained, and evaluated basic Machine Learning models for predictive analysis.

## Technologies Used
- Python
- Google Colab
- Jupyter Notebook
- Pandas & NumPy
- Scikit-Learn
- Matplotlib & Seaborn
